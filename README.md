# Skiplist
Templated Skip list implementation in C++ 

```c
Data structure specs:
Algorithm    Average    Worst Case
Space        O(n)       O(nlog(n))
Search       O(log(n))  O(n)
Insert       O(log(n))  O(n)
Delete       O(log(n))  O(n)
```
